#App overview
My team and I had to develop an inventory app for outdoor adventures.
The app was specifically made for people who like to camp and to manage their inventory digitally.


#App features
-3 Screens
-Allows the user to enter information and when a button is pressed it will show at the last screen in detail
-Each Screen has a button to the next screen
-3rd Screen has a back button
-Users enter there choices on the second screen which will then be calculated
